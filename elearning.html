QTI & LTI

Question and Test Interoperability (QTI) is IMS specification for assessment. It provides a data model for the representation of questions(items) and assignments and their outcome. It enables the exchange of questions and assignments between authoring tools and learning systems.

QAIT DevLabs uses QTI as a native format, i.e. items are stored in QTI. We don’t only import/export QTI, we store it natively and therefore item and test editors actually author QTI directly.

This means no information is “lost in translation” from a proprietary internal spec to or from QTI. We have been actively working within IMS and the QTI Technical Working groups to assure valid implementations and therefore assure interoperability

Learning Tools Interoperability(LTI) is a specification developed by IMS Global Learning Consortium. LTI is a way for one system (the tool consumer, typically an LMS) to send a user to another system (the tool provider, some service that integrates with the LMS) in a trusted way. This allows the user to be automatically signed in and directed to a specific course or module when the provider renders content.
